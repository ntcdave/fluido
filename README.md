# Simulação de Fluído com SDL2

Este projeto implementa uma simulação de fluído que acompanha o cursor do mouse, utilizando a biblioteca SDL2.

│   ├── renderer

│   │   └── renderer.hpp
│   └── utils
│       ├── vector.cpp
│       └── vector.hpp
├── include
│   └── SDL2
├── lib
├── CMakeLists.txt
└── README.md
